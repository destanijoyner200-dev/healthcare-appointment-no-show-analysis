# Healthcare Appointment No Show Analysis

This project analyzes healthcare appointment data to identify patterns in patient no shows and estimate the operational and financial impact on healthcare providers.

## Tools Used
SQL Server
Tableau Public
Excel

## Project Overview
Using SQL queries and Tableau visualizations, this project explores trends in missed healthcare appointments including diagnosis patterns, scheduling trends, patient demographics, and revenue impact.

## Key Insights
• 40% appointment no show rate identified in the dataset  
• Asthma and diabetes patients had the highest missed visit counts  
• Monday, Tuesday, and Thursday showed the highest no show frequency  
• Average revenue lost per missed appointment was approximately $240
